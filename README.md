# Minecraft-Websocket  

By XSuperMeteorX
* 如何安装   

`apt install wget git nodejs`
`git clone https://github.com/XSuperMeteorX/Minecraft-Websocket.git && cd Minecraft-Websocket`

复制这些代码需要<kbd>Ctrl</kbd>+<kbd>V</kbd>


* 如何启动    
`node main`

* 第二次\[之后的每一天\]启动    
`cd Minecraft-Websocket`
`node main`
